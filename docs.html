Aura Models Template

Documentation

Introduction

Welcome to the Aura Models HTML template, a premium one-page website designed for fashion and influencer agencies. This documentation guides you through setup, customization, and deployment.

Folder Structure

  • index.html: The main template file.
  • docs.html: This documentation file.
  • license.html: License agreement file.
  • styles.css: External CSS file for styling.
  • scripts.js: External JavaScript file for interactivity.
  • images/: Contains all images (e.g., a9.webp, a6.webp). Replace with your own images (WebP recommended).

Getting Started

1. Extract the template package.

2. Open index.html in a browser to preview.

3. Edit files using a code editor (e.g., VS Code).

4. Host on a web server (e.g., Netlify, GitHub Pages) for online access.

Customization

Customize the template by editing index.html, styles.css, and scripts.js:

  • Brand Name: Replace "Aura Models" in the logo, hero, and footer in index.html (search for Aura Models).
  • Images: Replace files in images/ with your own, keeping the same names (e.g., a9.webp).
  • Social Links: Update URLs in the settings object in scripts.js (e.g., https://facebook.com).
  • Contact Details: Update email, phone, and address in the contact section of index.html.
  • Styles: Modify styles.css for custom colors, fonts, or layouts.

Form Setup

The contact form uses a mock submission for demo purposes. For production:

  • Sign up at Formspree and get a form ID.
  • Replace the form action in index.html with your Formspree URL (e.g., https://formspree.io/f/your-form-id).
  • Alternatively, set up a custom backend (e.g., Node.js, PHP) and update the form action.

Troubleshooting

  • Images not loading: Ensure images are in the images/ folder with correct names.
  • Form not working: Verify the Formspree URL or backend setup.
  • Responsive issues: Test in modern browsers (Chrome, Firefox, Safari, Edge).
  • Scripts not working: Ensure scripts.js is in the root folder and linked correctly in index.html.
